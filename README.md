# 자동차 경주 게임

## 기능 구현 목록

### 자동차 (Car)
- [ ] 이름과 위치를 가진 객체 생성  
- [ ] 0~9 무작위 값이 4 이상이면 전진, 미만이면 정지  
- [ ] 위치는 `-` 기호로 표현  

### 입력 및 검증
- [ ] 자동차 이름: 5자 이하, 중복/공백 불가  
- [ ] 시도 횟수: 양수만 허용  

### 게임 진행 (RacingGame)
- [ ] 여러 자동차가 지정된 횟수만큼 이동  
- [ ] 각 라운드별 자동차 상태 출력  
- [ ] 최종 우승자(복수 가능) 출력  

### 예외 처리
- [ ] 잘못된 입력 시 `IllegalArgumentException` 발생
