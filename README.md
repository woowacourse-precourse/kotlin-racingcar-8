# 자동차 경주

> 초간단 자동차 경주 게임을 구현한다.

## 기능 목록

### 1. 입력

---

***1. 경주할 자동차 이름(이름은 쉼표(,) 기준으로 구분) 입력***

```
경주할 자동차 이름을 입력하세요.(이름은 쉼표(,) 기준으로 구분)
> pobi,woni,jun
```

***2. 시도할 횟수(이하 라운드라 칭함) 입력***

```
시도할 횟수는 몇 회인가요?
> 5
```

- 몇 번의 라운드를 진행할 지 **정수**로 입력

### 2. 출력

---

***1. 실행 결과 타이틀 출력***

```
실행 결과
```

***2. 라운드 별 실행 결과 출력***

```
pobi : -
woni : 
jun : -
```

- `${이름} : ${'-'.repeat(전진 횟수)}`
    - 전진 횟수만큼 `-`로 표시
    - 동일한 이름이 있는 경우 이름 뒤에 태그를 붙여 출력

***3. 우승자 출력***

```
최종 우승자 : ${이름}
```

- 우승자가 여러명인 경우 쉼표(,)로 구분하여 출력(입력 순)
    - `최종 우승자 : pobi, jun`
    - 동일한 이름이 있는 경우 이름 뒤에 태그를 붙여 출력
      - ex) `최종 우승자 : pobi#A, pobi#B`

### 3. 경주 게임

---

***1. 참가자 정의***

- 입력으로 주어진 이름을 쉼표로(,) 구분
- 각 이름은 1~5글자 이하만 가능
- 동일한 이름이 있는 경우 순서대로 `A, B, C...` 태그 부여
- 각 참가자의 초기 점수 = 0

***2. 라운드 진행(라운드가 끝날 때까지 반복)***

***2-1. 플레이어 별 전진, 멈춤 여부 결정***

- 각 플레이어에 대해 0~9 사이의 무작위 정수를 구함
    - `camp.nextstep.edu.missionutils.Randoms`의 `pickNumberInRange()`를 활용
    - 값이 **4 이상**일 경우: 전진(해당 플레이어 점수 +1)
    - 값이 **4 미만**일 경우: 멈춤(점수 변화 X)

***2-2. 라운드 결과 출력(출력-2 참고)***

***3. 우승자 판정***

- 점수가 가장 높은 플레이어들 찾기

## 예외 처리

1. *경주할 자동차 이름 입력 시*
    - 입력값이 비어있는 경우
    - 쉼표로 구분된 문자열이 비어있거나, 5자가 초과인 경우
2. 시도할 횟수입력 시
    - 입력값이 비어있는 경우
    - 입력값이 양의 정수가 아닌 경우(0, 음수, 정수가 아닌 유리수)

`IllegalArgumentException`을 발생시킨 후 애플리케이션 종료
