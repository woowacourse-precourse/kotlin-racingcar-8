# kotlin-racingcar-precourse

# 주요 기능

### 1. 자동차 이름 입력 처리 (readCarNames, validateCarNames)

- **기능**: 쉼표(,)로 구분된 자동차 이름을 입력받아 유효성을 검사하고 리스트로 반환합니다.
- **예시**: `"car1,car2,car3"` → `["car1", "car2", "car3"]`
- **처리 과정**
    1. 입력 문자열을 쉼표로 분리
    2. 각 이름의 공백 제거 및 빈 이름 필터링
    3. 이름 길이가 5자 이하인지 확인 (Car 인스턴스를 생성하면서 init으로 유효성 체크)
    4. 입력 형식 및 쉼표 개수 검증 (`쉼표 개수 +1 == 이름 개수` → `validateCarNames`)
- **예외 처리**
    - 빈 이름이 포함되거나 형식이 잘못된 경우: "빈 이름이 있거나 입력 형식이 잘못되었습니다."
    - 이름이 5자를 초과하는 경우: "이름은 5자 이하여야 합니다."
- **메서드**: readCarNames(): List<String>, validateCarNames(names: List<String>, input: String)

---

### 2. 시도 횟수 입력 처리 (readTryCount, validateTryCount)

- **기능**: 시도 횟수를 입력받아 자연수인지 검증하고 반환합니다.
- **예시**: `"3"` → `3`
- **처리 과정**
    1. 입력 문자열을 정수로 변환
    2. 자연수 여부 검증 (`validateTryCount`)
- **예외 처리**
    - 입력이 자연수가 아닌 경우: "시도 횟수는 자연수여야 합니다."
- **메서드**: readTryCount(): Int, validateTryCount(tryCount: Int?)

---

### 3. 자동차 상태 관리 (Car 클래스)

- **기능**: 자동차의 이름과 점수를 관리하며, 진행 상황을 출력하거나 상태를 조회합니다.
- **세부 메서드**
    1. **이동 처리** (move)
        - **기능**: 무작위 숫자(0~9)에 따라 자동차가 전진 여부를 결정합니다.
        - **처리 과정**: 숫자가 4 이상이면 점수 1 증가
        - **메서드**: move(randomNumber: Int)
    2. **진행 상황 출력** (showCurrentProgress)
        - **기능**: 자동차의 현재 진행 상황을 -로 표시해 출력합니다.
        - **예시 출력**: `car1 : ---` (점수가 3인 경우)
        - **메서드**: showCurrentProgress()
    3. **점수 조회** (getScore)
        - **기능**: 현재 자동차의 점수를 반환합니다.
        - **메서드**: getScore(): Int
    4. **이름 조회** (getName)
        - **기능**: 자동차의 이름을 반환합니다.
        - **메서드**: getName(): String

---

### 4. 게임 진행 (Game 클래스)

- **기능**: 한 라운드를 진행하며 모든 자동차를 이동시키고 현재 진행 상황을 출력합니다.
- **세부 메서드**
    1. **라운드 진행** (play)
        - **기능**: 한 라운드를 실행하여 모든 자동차를 이동시키고 진행 상황을 출력합니다.
        - **처리 과정**
            - 모든 자동차를 이동 (moveCars 호출)
            - 각 자동차의 진행 상황 출력 (showAllCarsProgress 호출)
        - **메서드**: play()
    2. **자동차 이동 처리** (moveCars)
        - **기능**: 모든 자동차에 대해 무작위 숫자를 생성하여 이동 여부를 결정합니다.
        - **처리 과정**: 각 자동차에 대해 `Randoms.pickNumberInRange(0,9)`를 호출하여 move 메서드 실행
        - **메서드**: moveCars()
    3. **진행 상황 출력** (showAllCarsProgress)
        - **기능**: 모든 자동차의 현재 진행 상황을 출력합니다.
        - **처리 과정**: 각 자동차의 showCurrentProgress 호출
        - **메서드**: showAllCarsProgress()

---

### 5. 우승자 결정 및 출력 (Game 클래스)

- **기능**: 게임 종료 후 최고 점수를 획득한 자동차 리스트를 우승자로 출력합니다.
- **세부 메서드**
    1. **최고 점수 계산** (getBestScore)
        - **기능**: 모든 자동차의 점수 중 최대값을 반환합니다.
        - **처리 과정**: `cars.maxOf { it.getScore() }`를 사용해 최대 점수 계산
        - **메서드**: getBestScore(): Int
    2. **우승자 출력** (showWinners)
        - **기능**: 최고 점수를 보유하고 있는 자동차 리스트를 출력합니다.
        - **처리 과정**
            - 최대 점수를 기준으로 자동차 필터링
            - 해당 자동차의 이름을 쉼표로 구분해 출력
        - **예시 출력**: `최종 우승자: aaa, bbb`
        - **메서드**: showWinners()
