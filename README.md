# kotlin-racingcar-precourse

## 구현할 기능 목록

1. 입력 기능

- 자동차 이름 입력받기
- 시도 횟수 입력받기

2. 입력 검증 기능

- 자동차 이름이 빈 값인지 검증
- 자동차 이름이 5자를 초과하는지 검증
- 자동차 이름에 중복이 있는지 검증
- 시도 횟수가 숫자인지 검증
- 시도 횟수가 양수인지 검증

3. 자동차 생성 기능

- 쉼표(,)를 기준으로 자동차 이름 분리
- 각 자동차 이름으로 자동차 객체 생성
- 자동차 초기 위치는 0으로 설정

4. 게임 진행 기능

- 0~9 사이의 무작위 값 생성
- 무작위 값이 4 이상이면 자동차 전진
- 무작위 값이 4 미만이면 자동차 정지
- 주어진 횟수만큼 라운드 반복

5. 출력 기능

- 각 라운드마다 실행 결과 출력
- 자동차 이름과 위치를 "-"로 표시하여 출력
- 라운드 사이에 빈 줄 출력

6. 우승자 판정 기능

- 모든 자동차 중 최대 이동 거리 찾기
- 최대 이동 거리를 가진 자동차 찾기
- 우승자가 여러 명인 경우 쉼표(,)로 구분하여 출력
