# 자동차 경주

## 구현할 기능 목록
### 1. 사용자에게 입력 받기
- [x] 경주할 자동차 이름 입력 받기
- [x] 경주 시도할 횟수 입력 받기

### 2. 입력 받은 문자열 중 자동차 이름 추출하기
- [x] `,` 기준으로 자동차 이름 추출하기
- [ ] 추출된 자동차 이름의 길이가 5자 이하인지 확인하기
  - [ ] 이름이 6자 이상일 경우 `IllegalArgumentException` 발생시키기

### 3. 자동차 경주 시작
- [ ] 자동차 별 진행 상황 저장하기
- [ ] 라운드가 진행될 때 마다 자동차 별로  `0-9` 사이 무작위 수를 생성하기
- [ ] 자동차 별로 생성된 무작위 수 중 `4` 이상인 자동차는 한 칸 전진시키기
- [ ] 라운드가 끝날때마다 자동차 별 진행 상황 업데이트 하기
- [ ] 진행 상황 업데이트 후 진행 상황 출력하기

### 4. 자동차 경주 종료
- [ ] 가장 많이 전진한 자동차 선정하기
  - [ ] 동률일 경우 공동 우승자로 선정하기
- [ ] 최종 우승자 출력하기
