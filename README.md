# kotlin-racingcar-precourse

## 기능 목록

### 1. 입력 기능
- [x] 자동차 이름 입력받기
  - 쉼표(,)로 구분된 자동차 이름 문자열 입력
- [x] 시도 횟수 입력받기
  - 정수 형태의 시도 횟수 입력

### 2. 입력 검증 기능
- [x] 자동차 이름 검증
  - 각 자동차 이름이 5자 이하인지 확인
  - 자동차 이름이 빈 문자열이 아닌지 확인
- [x] 시도 횟수 검증
  - 시도 횟수가 양수인지 확인
  - 시도 횟수가 숫자인지 확인

### 3. 자동차 도메인 기능
- [x] 자동차 생성
  - 이름과 초기 위치(0)를 가진 자동차 생성
- [x] 자동차 전진
  - 무작위 값이 4 이상일 때 위치 증가
- [x] 자동차 위치 조회
  - 현재 위치 반환

### 4. 경주 게임 기능
- [x] 자동차 경주 게임 생성
  - 여러 대의 자동차로 경주 게임 구성
- [x] 라운드 진행
  - 각 자동차마다 무작위 값 생성(0~9)
  - 4 이상이면 전진, 미만이면 정지
- [x] 우승자 결정
  - 가장 많이 전진한 자동차 찾기
  - 공동 우승자 처리

### 5. 출력 기능
- [x] 라운드별 진행 상황 출력
  - 각 자동차의 이름과 현재 위치를 `-`로 표시
- [x] 최종 우승자 출력
  - 단독 우승자 또는 공동 우승자를 쉼표(,)로 구분하여 출력

### 6. 예외 처리
- [x] 잘못된 입력 시 IllegalArgumentException 발생
  - 자동차 이름이 5자를 초과하는 경우
  - 자동차 이름이 빈 문자열인 경우
  - 시도 횟수가 양수가 아닌 경우
  - 시도 횟수가 숫자가 아닌 경우
