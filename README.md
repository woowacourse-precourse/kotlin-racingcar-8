# 자동차 경주 게임

## 기능 구현 목록

### 자동차 (Car)
- [x] 이름과 위치를 가진 객체 생성
- [x] 0~9 무작위 값이 4 이상이면 전진, 미만이면 정지
- [x] 위치는 `-` 기호로 표현
- [x] 자동차 이동 기능 단위 테스트 작성
- [x] 위치 표시 기능 단위 테스트 작성

### 입력 및 검증
- [x] 자동차 이름: 5자 이하, 중복/공백 불가
- [x] 시도 횟수: 양수만 허용
- [x] 자동차 이름 검증 예외 테스트 작성
- [x] 시도 횟수 검증 파라미터화 테스트 작성

### 게임 진행 (RacingGame)
- [x] 여러 자동차가 지정된 횟수만큼 이동
- [x] 각 라운드별 자동차 상태 출력
- [x] 최종 우승자(복수 가능) 출력
- [x] 게임 라운드 진행 단위 테스트 작성
- [x] 우승자 판별 단위 테스트 작성
- [x] 공동 우승자 처리 테스트 작성

### 예외 처리
- [x] 잘못된 입력 시 `IllegalArgumentException` 발생
- [x] 통합 테스트로 예외 발생 검증

## 테스트 구성
- **CarTest**: 자동차 이동 및 위치 표시 테스트
- **RacingGameTest**: 게임 진행 및 우승자 판별 테스트
- **InputValidatorTest**: 입력 검증 및 예외 처리 테스트
- **ApplicationTest**: 전체 기능 통합 테스트
