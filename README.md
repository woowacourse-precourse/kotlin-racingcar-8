# 자동차 경주

## 구현하면서 특별히 집중할 부분
### 🖥️ 쉽게 이해가 되는 코드 만들기 🖥️
### 1. 변수명, 함수명, 클래스명 네이밍
- 다른 사람이 코드를 볼 때, 별도의 주석 없이도 이름만 보고 무슨 기능을 하는지 이해할 정도로 네이밍하기

### 2. 함수 길이 최소화
- 함수 당 코드의 길이가 20줄이 넘지 않도록 구현하기

## 구현할 기능 목록
### 1. 사용자에게 입력 받기
- [x] 경주할 자동차 이름 입력 받기
- [x] 경주 시도할 횟수 입력 받기
- [x] 경주 시도할 횟수가 정상적인 입력값인지 검증하기
  - [x] 0 이하의 수일 경우 `IllegalArgumentException` 발생
  - [x] 숫자가 아닐 경우 `IllegalArgumentException` 발생

### 2. 입력 받은 문자열 중 자동차 이름 추출하기
- [x] `,` 기준으로 자동차 이름 추출하기
- [x] 추출된 자동차 이름의 길이가 1자 이상, 5자 이하인지 확인하기
  - [x] 1자 이상, 5자 이하의 범위를 벗어날 경우 `IllegalArgumentException` 발생시키기

### 3. 자동차 경주 시작
- [x] 각각의 자동차 이름으로 자동차 객체 생성하기
- [x] 자동차 별로 생성된 무작위 수 중 `4` 이상인 자동차는 한 칸 전진시키기
- [x] 진행 상황 업데이트 후 진행 상황 출력하기

### 4. 자동차 경주 종료
- [x] 가장 많이 전진한 자동차 선정하기
  - [x] 동률일 경우 공동 우승자로 선정하기
- [x] 최종 우승자 출력하기

## 스스로 판단한 구현사항
### 1. 2개 이상의 동일한 차량 이름이 등록될 때
- 동일한 차량 이름이 2개 이상일 경우, 입력된 순서대로 A,B,C의 접미사를 이름 뒤에 붙인다.
- 접미사는 이름 길이 제한을 무시한다. 예를 들어 `Calls` 이름이 2개 입력될 경우, `CallsA` , `CallsB`로 이름을 변경한다.
