# kotlin-racingcar-precourse
## 기능 요구 사항 정리

### Controller (RacingController)

- 사용자 입력을 받아서 자동차 이름 목록과 시도 횟수를 전달받는다.
- 입력값의 유효성을 검사한다.
- `RacingGame`을 실행시켜 시도 횟수만큼 자동차들을 움직인다.
- 각 라운드별 결과를 출력한다.
- 게임이 종료되면 `WinnerCalculator`를 통해 우승자를 계산하고 출력한다.

---

### View

### InputView

- 자동차 이름을 입력받는다.
    - 쉼표(,) 기준으로 구분하며, 이름이 비어있거나 5자를 넘어가면 예외 발생
- 시도할 횟수를 입력받는다.
    - 숫자가 아닐 경우 예외 발생

### OutputView

- 각 시도 결과(자동차별 이동 상태)를 출력한다.
- 우승자 목록을 쉼표(,)로 구분하여 출력한다.
- 현재 위치만큼 `"-"`로 표시하여 출력한다.

---

### Domain

### Car

- 자동차 이름과 현재 위치를 저장한다.
- 이름 길이가 5자 초과이거나 이름이 비어있으면 예외 발생
- 0~9 사이의 랜덤 숫자를 받아 4 이상일 때 한 칸 전진한다.


### RacingGame

- 모든 자동차 리스트를 관리한다.
- 주어진 시도 횟수만큼 각 자동차를 전진시킨다.

### WinnerCalculator

- 가장 멀리 간 자동차의 위치를 구한다.
- 해당 위치에 도달한 자동차가 여러 대일 경우 공동 우승 처리한다.

---

## 테스트 항목

- **CarTest** : 자동차 이름 및 전진 조건 검증  
- **RacingGameTest** : 자동차 리스트 생성 및 이동 로직 검증  
- **WinnerCalculatorTest** : 우승자 및 공동 우승자 계산 검증  
- **InputViewTest** : 입력값 유효성(빈 이름, 길이 초과) 검증  
- **OutputViewTest** : 출력 형식 및 우승자 출력 검증  
- **ApplicationTest** : 프로그램 통합 테스트 (정상 흐름 & 예외 케이스)
