# kotlin-racingcar-precourse

## 기능 목록 (Feature List)

### 입력 관련
- 경주할 자동차 이름을 입력받는다.
    - 쉼표(`,`)로 구분한다.
    - 각 이름은 공백을 제거하고 **1자 이상 5자 이하**여야 한다.
    - 잘못된 입력(빈 문자열, 6자 이상 등)일 경우 `IllegalArgumentException`을 발생시키고 프로그램을 종료한다.

- 시도할 횟수를 입력받는다.
    - 양의 정수만 허용한다.
    - 숫자가 아니거나 1 미만일 경우 `IllegalArgumentException`을 발생시키고 프로그램을 종료한다.

---

### 게임 로직
- 각 자동차는 이름과 현재 위치(전진 거리)를 가진다.
- 시도 횟수만큼 반복하며 각 라운드를 수행한다.
- 매 라운드마다 각 자동차는 0~9 사이의 무작위 숫자를 뽑는다.
    - 해당 숫자가 **4 이상일 경우 전진**한다.
- 전진할 때마다 `"-"`를 추가하여 이동 거리를 표시한다.

---

### 출력
- 각 라운드가 끝날 때마다 모든 자동차의 현재 상태를 출력한다.  
  pobi : --
  woni : ---
  jun : -

markdown
코드 복사
- 모든 라운드가 종료된 후 최종 우승자를 출력한다.
- 가장 많이 전진한 자동차가 우승자이다.
- 우승자가 여러 명일 경우 쉼표(`,`)로 구분한다.  
  최종 우승자 : pobi, jun
