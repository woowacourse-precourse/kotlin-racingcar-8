# kotlin-racingcar-precourse





\## 목표

초간단 자동차 경주 게임을 구현한다.





\### 기능 목록



\- \[ O ] \*\*\[입력]\*\* 경주할 자동차 이름을 입력 받는다. (이름은 쉼표(`,`) 기준으로 구분)

\- \[ O ] \*\*\[입력]\*\* 시도할 횟수를 입력 받는다.

\- \[ O ] \*\*\[검증]\*\* 사용자가 잘못된 값을 입력할 경우 `IllegalArgumentException`을 발생시키고 애플리케이션을 종료한다.

&nbsp; - 자동차 이름이 \*\*5자를 초과\*\*하는 경우

&nbsp; - 자동차 이름이 \*\*비어있거나 공백\*\*인 경우 (예: `pobi,,jun` 또는 `pobi, ,jun`)

&nbsp; - 시도할 횟수가 \*\*유효한 숫자가 아닌\*\* 경우

\- \[ O ] \*\*\[로직]\*\* 0에서 9 사이의 무작위 값을 구한 후, 이 값이 4 이상일 경우 자동차는 전진한다.

\- \[ ] \*\*\[로직]\*\* 입력받은 시도 횟수만큼 4번 항목(전진/멈춤)을 반복 실행한다.

\- \[ ] \*\*\[출력]\*\* 각 실행 턴(round)마다 자동차들의 현재 위치를 출력한다. (예: `pobi : --`)

\- \[ ] \*\*\[출력]\*\* 경주가 완료된 후, 최종 우승자를 출력한다.

&nbsp; - 우승자가 여러 명일 경우 쉼표(`,`)로 구분하여 출력한다. (예: `최종 우승자 : pobi, jun`)
