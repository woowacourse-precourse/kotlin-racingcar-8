# 자동차 경주

## 구현 기능 목록

- 경주할 자동차 이름들을 입력받는다.
- 시도 횟수를 입력받는다.
- 각 자동차는 0~9 사이의 랜덤값을 얻는다.
- 4 이상이면 전진한다.
- 시도 횟수만큼 반복하며 각 차수의 결과를 저장한다.
- 가장 많이 전진한 자동차(들)을 우승자로 판단한다.
- 각 차수별 실행 결과를 출력한다.
- 우승자(들)을 출력한다.

## 💡 예외 처리 방식

- 이름이 비어 있거나 5자를 초과하면 예외 발생.
- 시도 횟수가 숫자가 아니거나 1 미만이면 예외 발생.
- 잘못된 입력 시 `IllegalArgumentException` 발생 후 프로그램 종료.
