# kotlin-racingcar-precourse

# 문제 분석
이름을 입력 받은 n대의 자동차가 자동차 경주를 진행한다.
이동에 대한 횟수를 입력받고, 조건에 해당한다면 자동차가 이동한다.
우승자를 분석하고, 출력한다.

# 개발 진행 과정
1. 자동차의 이름과 이동 범위에 대한 클래스를 생성한다.
2. 각 자동차에 대한 이름과 이동 횟수를 입력받고, 잘못된 값의 경우 오류를 반환한다.
3. 조건에 맞추어 이동하는 함수를 생성한다.
4. 이동 횟수를 비교하고, 우승자를 찾는 함수를 생성한다.
5. 이동을 n회 진행하고, 우승자를 출력한다.

# 유의 사항
- indent depth가 3이 넘지 않도록 한다.
- 함수는 한 가지 일만 하도록 최대한 작게 만든다.
- camp.nextstep.edu.missionutils에서 제공하는 API를 사용해 구현한다.
- JUnit 5와 AssertJ을 이용해 테스트 코드를 확인한다.
