# 주요 기능 설명

### main()
- 전체 게임 실행 흐름 제어

### PlayerNameInputErrorCheck()
- 이름 입력 검증
- 검증내용: 5자를 넘는지, 이름에 공백이 들어가는지

### FindCurrentMostMove()
- 각 라운드에서 가장 많이 이동한 거리 계산

### PrintRacingProcess()
- 각 라운드의 경주 상태 출력

### PlayerForward()
- 난수를 통해 주어진 조건과 맞으면 자동차 이동

### FindRacingWinner()
- 최종 우승자를 판단하는 기능

### RecordRacingWinner()
- 한명 또는 여러명의 우승자를 처리하고, 문자열로 반환
