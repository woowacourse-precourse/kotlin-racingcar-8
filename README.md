# kotlin-racingcar-precourse

## 구현할 기능 목록

- [ ] 자동차 이름이 중복되는 경우 예외로 처리한다.
- [ ] 자동차 이름은 1 글자 이상의 문자를 포함해야 한다.
    - 빈 문자열(`""`)은 예외로 처리한다.
- [ ] 자동차 이름이 5글자 이하인지 확인한다.
- [ ] 몇 번의 이동(시도 횟수)을 할 것인지 입력받는다.
- [ ] 시도 횟수가 0이라면 우승자를 출력 후 게임을 종료한다.
- [ ] 시도 횟수가 음수라면 예외로 처리한다.
- [ ] 시도 횟수가 단일 숫자가 아니라면 예외로 처리한다.
- [ ] 시도 횟수가 최대 데이터 범위를 벗어나는 경우 예외로 처리한다.
- [ ] 0-9사이 무작위 정수가 4이상이라면 자동차를 한 칸 전진시킨다.
- [ ] 사용자가 입력한 시도 횟수 만큼 전진 시키거나 멈춘다(nop).
- [ ] 자동차 게임의 완료 여부를 확인한다.
- [ ] 게임이 완료되었다면 우승자를 출력한다.
    - 우승자가 단독인 경우
    - 우승자가 여러명인 경우

## 결정한 요구 사항

### 1. 자동차 이름의 중복을 허용하지 않는다.

자동차 이름이 중복되면 누가 어느 자동차인지 구분이 어려워 결과 해석이 모호해집니다.
따라서 이름 중복을 허용하지 않고 예외로 처리했습니다.

### 2. 사용자의 입력 시도는 0이상이다.

음수는 예외로 처리했습니다.
0의 경우 한 번도 시도하지 않는다는 의미로 유효하다고 판단했습니다.(바로 우승자 출력)
