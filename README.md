# kotlin-racingcar-precourse

## 기능 목록

### 입력 기능
- [x] 자동차 이름을 입력받는다.
  - 쉼표(,)를 기준으로 구분한다.
- [x] 시도할 횟수를 입력받는다.

### 유효성 검증 기능
- [x] 자동차 이름이 5자 이하인지 검증한다.
  - 5자를 초과하면 IllegalArgumentException을 발생시킨다.
- [x] 자동차 이름이 비어있지 않은지 검증한다.
  - 빈 이름이 있으면 IllegalArgumentException을 발생시킨다.
- [x] 시도 횟수가 양수인지 검증한다.
  - 0 이하의 값이면 IllegalArgumentException을 발생시킨다.

### 자동차 경주 게임 로직
- [x] 0에서 9 사이의 무작위 값을 생성한다.
- [x] 무작위 값이 4 이상이면 자동차를 전진시킨다.
- [x] 무작위 값이 4 미만이면 자동차를 멈춘다.
- [x] 각 자동차의 현재 위치를 관리한다.
- [x] 주어진 횟수만큼 경주를 진행한다.

### 출력 기능
- [x] 각 시도마다 자동차의 이름과 위치를 출력한다.
  - 위치는 `-` 기호로 표현한다.
- [x] 실행 결과를 라운드별로 출력한다.
- [x] 최종 우승자를 출력한다.
  - 우승자가 여러 명일 경우 쉼표(,)로 구분하여 출력한다.

### 우승자 판정 기능
- [x] 가장 많이 전진한 자동차를 찾는다.
- [x] 최대 위치에 있는 자동차가 여러 대인 경우 모두 우승자로 선정한다.
