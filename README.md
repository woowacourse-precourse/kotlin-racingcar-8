# kotlin-racingcar-precourse
> 과제 세부 요구 사항은 [ASSIGNMENT.md](./ASSIGNMENT.md)를 참고해주세요

## 구현할 기능 목록
- [ ] 자동차 이름 입력
- [ ] 자동차 이름들 유효성 검사
  - 빈값, 글자 수 5자 이하
- [x] 자동차 DTO 설계
- [x] 자동차 객체 생성
- [ ] 시도 횟수 입력
- [x] 시도 횟수 만큼 반복 로직 구성
  - [x] 0-9 사이 무작위 값 입력
  - [x] 무작위 값이 4 초과 시 자동차 전진
  - [x] 각 차수별 경주 상황 출력
- [ ] 최종 우승자 출력
  - [ ] 우승자가 복수일 경우 모두 출력
