# kotlin-racingcar-precourse
### 구현 할 기능 목록 정리
#### 1. Car 도메인 클래스 정의
- 자동차의 이름과 현재 이동 거리(위치)를 상태로 가지는 Car 클래스를 구현한다.
- Car 객체 생성 시 위치는 0으로 초기화된다.

#### 2. Car 이동 로직 구현
- Car 클래스 내부에 전진/멈춤을 결정하는 로직을 구현한다.
- 0에서 9 사이의 무작위 값을 생성하며, 값이 4 이상일 경우에만 위치를 1 증가시킨다.

#### 3. 사용자 입력 기능 구현
- 사용자로부터 자동차 이름(쉼표 구분)과 시도 횟수를 입력받는 기능을 구현한다. (Console API 사용)

#### 4. 입력값 유효성 검증
- 입력된 자동차 이름이 5자 이하인지 검증한다. 입력된 시도 횟수가 유효한 양의 정수인지 검증한다.
- 유효성 검증 실패 시 IllegalArgumentException을 발생시킨다.

#### 5. 게임 실행 관리 및 라운드 반복
- 입력받은 횟수만큼 반복을 제어하며 게임을 실행하는 메인 흐름을 구현한다.
- 함수를 분리하여 모든 로직의 indent depth를 2 이하로 유지하는 기본 구조를 마련한다.

#### 6. 단일 라운드 실행 및 차수별 결과 출력
- 한 번의 라운드에서 모든 자동차의 이동 로직을 실행하고 그 결과를 출력하는 기능을 구현한다.
- 매 차수가 끝날 때마다 모든 자동차의 이름과 현재 이동 거리를 하이픈(-)으로 시각화하여 출력한다.

#### 7. 최종 우승자 선정 및 결과 출력
- 경주 완료 후, 모든 자동차 중 최대 이동 거리를 구하고 해당 거리를 가진 자동차(들)을 우승자로 선정한다.
- 우승자 이름을 쉼표(,)로 구분하여 "최종 우승자 : 이름1, 이름2" 형식으로 출력한다.
