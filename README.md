# 자동차 경주

## 구현할 기능 목록
### 1. 사용자에게 입력 받기
- [x] 경주할 자동차 이름 입력 받기
- [x] 경주 시도할 횟수 입력 받기
- [x] 경주 시도할 횟수가 정상적인 입력값인지 검증하기
  - [x] 0 이하의 수일 경우 `IllegalArgumentException` 발생
  - [x] 숫자가 아닐 경우 `IllegalArgumentException` 발생

### 2. 입력 받은 문자열 중 자동차 이름 추출하기
- [x] `,` 기준으로 자동차 이름 추출하기
- [x] 추출된 자동차 이름의 길이가 5자 이하인지 확인하기
  - [x] 이름이 6자 이상일 경우 `IllegalArgumentException` 발생시키기

### 3. 자동차 경주 시작
- [x] 각각의 자동차 이름으로 자동차 객체 생성하기
- [x] 자동차 별로 생성된 무작위 수 중 `4` 이상인 자동차는 한 칸 전진시키기
- [x] 진행 상황 업데이트 후 진행 상황 출력하기

### 4. 자동차 경주 종료
- [x] 가장 많이 전진한 자동차 선정하기
  - [x] 동률일 경우 공동 우승자로 선정하기
- [ ] 최종 우승자 출력하기
