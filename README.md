# kotlin-racingcar-precourse

## 🎯 기능 목록

- [x] 경주할 자동차 이름을 입력받는다.
    - [x] 이름은 쉼표(,)를 기준으로 구분한다.
    - [x] 이름은 5자 이하만 가능하다.
    - [x] 잘못된 이름 입력 시 `IllegalArgumentException`을 발생시킨다.
- [ ] 시도할 횟수를 입력받는다.
    - [ ] 입력값은 숫자여야 한다.
    - [ ] 잘못된 횟수 입력 시 `IllegalArgumentException`을 발생시킨다.
- [ ] 입력된 횟수만큼 경주를 진행한다.
    - [ ] 각 자동차는 0에서 9 사이의 무작위 값을 기준으로 4 이상일 경우 전진(1칸)한다.
    - [ ] 각 차수의 실행 결과를 출력한다. (예: `pobi : --`)
- [ ] 경주가 끝난 후 최종 우승자를 결정한다.
    - [ ] 가장 많이 전진한 자동차가 우승한다.
    - [ ] 공동 우승자가 있을 수 있다.
- [ ] 최종 우승자를 출력한다.
    - [ ] 공동 우승 시 쉼표(,)로 구분하여 출력한다. (예: `최종 우승자 : pobi, jun`)
