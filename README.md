# kotlin-racingcar-precourse

## **자동차 경주** ## 

## 초간단 자동차 경주 게임을 구현한다.

---

## **구현 할 기능 목록**

### 1.입력
    1-1 자동차의 이름 입력

    1-2 자동차의 이동 입력

---

### 2.자동차 경주
    2-1 자동차는 전진할수 있다.  

    2-2 자동차는 멈출수 있다.
---
### 3. 우승자 

* ### 우승자는 한명 이상일 수 있다. 

---

### 4. 출력
    4-1 입력 안내문 

    4-2 차수 마다 각 자동차의 이동 결과
        - 자동차 이름과 이동 결과를 같이 출력한다.  

    4-3 경주가 끝난후 우승자 
        - 우승자가 여러명일 경우 쉼표(,)를 이용하여 구분한다. 


---

### 5. 예외처리 

* ### 사용자가 잘못된 값을 입력할 경우 IllegalArgumentException을 발생시킨후 애플리케이션은 종료되어야 한다.

---
